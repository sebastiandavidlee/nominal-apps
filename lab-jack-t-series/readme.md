# Connect :: LabJack

This is intended to show Connect funcationlity with Labjack T-series of DAQ products.

To request U-series support, please reach out to Nominal.

## System Dependencies
Lists dependencies required of the system that Connect is installed onto.

1. The [LJM Library](https://support.labjack.com/docs/ljm-software-installer-downloads-t4-t7-t8-digit) from LabJack.

    **Windows Users**: Nominal recommends the [Minimal LJM Installer](https://support.labjack.com/docs/minimal-ljm-installers) since Connect will be the primary interaction with the LabJack unit.
